NAME

Hub::Data::File - File

Top

SYNOPSIS

  use Hub qw(:standard);
  my $file = mkinst('File', 'foo.txt');
  print ${$file->get_content()};

Top

DESCRIPTION

A basic file which can be written and saved. It acts as a singleton and is integrated into the filecache system (supports refresh).

Intention

The the FileSystem::snapshot manpage method creates an image of the file system. Each file is an instance of this class. We delay reading from disk until it is necessary. We do not keep a file handle open.
